# Samurai Portfolio — Adarsha Karki

A personal portfolio website built with HTML5, CSS3 and JavaScript as part of the final assignment of Internet Technology module at The British College, Kathmandu (UWE Bristol).

## Live Site

Hosted on GitHub Pages: [https://ads4331x.github.io/itassessement2026](https://ads4331x.github.io/itassessement2026)

## Pages

| Page     | File            | Description                                               |
| -------- | --------------- | --------------------------------------------------------- |
| Home     | `index.html`    | Hero slider, about summary, featured projects, cool sites |
| About    | `about.html`    | Bio, personal details table, interests                    |
| Skills   | `skills.html`   | Animated skill bars, skills table, toolbox                |
| Projects | `projects.html` | Filterable grid of 20+ GitHub repos                       |
| Gallery  | `gallery.html`  | Project screenshots with lightbox viewer                  |
| Resume   | `resume.html`   | Education, work experience, skills summary                |
| Contact  | `contact.html`  | Validated contact form, FAQ section                       |

## Features

- Responsive layout with CSS Grid and Flexbox
- Image slider with auto-advance and manual controls
- Animated skill bars triggered on scroll (IntersectionObserver)
- Contact form with JavaScript validation
- Photo gallery with lightbox (keyboard + click to close)
- Filter bar to sort projects by language
- Header and footer loaded dynamically via `fetch()`
- Tables with `colspan`, `rowspan` and nested tables
- Persistent navigation, local navigation and secondary footer nav
- Japanese samurai theme with consistent colour palette

## Tech Stack

- HTML5
- CSS3 (custom properties, Grid, Flexbox, animations)
- Vanilla JavaScript
- Fonts: Electroharmonix, Crimson Text (Google Fonts)
